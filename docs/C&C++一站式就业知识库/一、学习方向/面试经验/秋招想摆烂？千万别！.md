# 秋招想摆烂？ 千万别！

原文链接：[秋招想摆烂？ 千万别！_牛客网](https://www.nowcoder.com/feed/main/detail/e305b36cab14454aa57473e8387fa599?sourceSSR=search)

 随着互联网提前批结束正式批开始，秋招慢慢进入下一阶段，已经有一些人准备开始摆烂了，但是过来人经验告诉你   千万别！   国央企秋招刚刚开始罢了！很多人不知道秋招的重要性，一旦秋招不努力只剩下春招一次机会，而春招大部分人都会被论文给纠缠住，可能春招收获会更少更少，所以在最后的机会一定要继续海投海笔多面！可能到目前互联网的hc慢慢见底了，但是该投还是要投，因为肯定会有释放的人，投了还有可能成为被捞的人，不投可能一点机会也没了。     
  
可能绝大部分人在接下来一段时间会专心于国企央企的投递，但是大家也一定要确定好自己的初心。如果不喜欢写代码就安心国央企，如果想搞技术那国央企只能说作为避免自己失业的一根救命稻草，还是要专心寻找互联网无论大厂中厂还是小厂。  
  
此外，投递国央企时一定要好好选择，经验来看工资高的国央企除了电网和烟草都会加班，甚至部分加班程度不输互联网，而且单休的企业大有人在。特别奉劝大家打探好企业氛围，能否保证双休加班程度自己能不能接受，不要进来之后再后悔。而且特别注意，部分军工企业以及研究所较为严格，据了解加班程度能达到晚上10点，在选择之前一定要确定好自己能否接受。  
  
最后，说一些心里话。很理解大家大多因为就业不理想希望找个国央企轻松度日，所以在选择时候一定要好好考虑钱和轻松程度的权衡。钱不可能白挣，注定会跟你的工作时长挂钩，所以有时候活轻松钱较少的工作不一定不好，全看个人选择。警惕一些假国企，虽国有控股但可能跟大家想象的电网那种国央企差距特别大，不是所有的国企工作都是容易接受的。  
  
找工作这件事会伴随大家直到毕业，希望大家能把招聘的一年坚持到最后。选择最适合自己的也是自己愿意去干的。过来人告诉大家没有最好的工作，只有自己能接受的工作，适当降低职业期待。双休真的很重要！慎重选择！
