我这里讲的军工或科研单位，只针对国企或者事业单位，不适合民营军工行业。



因为这个话题比较敏感，<font style="color:#DF2A3F;">而且不同的单位情况也不完全一样，</font>建议签协议前多网上搜集下信息。

三航：航空、航天、航海  这些单位经费明显 比 兵器工业下辖的企业多，所以三航的待遇和发展空间也更大。

<font style="color:rgb(25, 27, 31);">军工所有一个月到手+公积金2万多的，也有年终奖十多万的。也有一个月到手四五千的，年终奖一两万，还单休的。  </font>**<font style="color:rgb(25, 27, 31);">所以也不能一概而论。</font>**



# 军工或科研单位的优势
1. 部分学校的专业可以作为保底
2. 工作比较稳定和有保障，发展也稳定，不太用担心裁员的问题
3. 待遇水平在当地平均能到中上





# 军工或科研单位的劣势
1. 部分科研、军工单位在二三线城市；

2. 加班也多，经常要出差，有些班是不加也得加，如果在民营大不了老子离职，但在军工或科研单位，不是想离职就马上离职；

3. 晋升通道相对单一和不好跳槽，在<font style="color:rgb(25, 27, 31);">军工是稳定， 但对个人来说发展后劲不足，提高薪水的途径就是一个，熬资历。</font>如果不想在军工待了再跳去民营比较困难。



# 建议
这个话题比较敏感，不敢建议，有冲劲建议冲击互联网，想安稳度日，可以考虑军工或者科研单位。
