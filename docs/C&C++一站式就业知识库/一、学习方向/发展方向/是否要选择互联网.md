# 是否要选择互联网

作者：[程序员老廖](https://space.bilibili.com/3494351095204205)



互联网是一个充满机遇行业（有可能赚到大钱），但也是一个竞争激烈充满挑战的行业（996是常态）。

所以建议根据自身情况进行选择。

# 互联网的优势
1. 互联网薪资待遇高、岗位种类也多；
2. 互联网企业 人**际关系**比传统企业**更为简单**，不需要花太多精力去处理人情世故；
3. 互联网上升渠道明朗，**只要技术牛逼**以及情商达到普通水平**也能往技术专家发展**，（在传统企业技术牛逼可能还没有情商 在线重要）
4. 无限发展潜力：互联网行业以其高速发展和创新的特点吸引了大量人才。随着数字化、智能化时代的到来，互联网行业将持续蓬勃发展，并在各个领域创造新的商业模式和就业机会。
5. 创业机会多：互联网行业给了普通人创业的机会。相较于传统行业，互联网行业门槛较低，创业成本相对较少，同时市场也更加开放。因此，想要实现自己的创业梦想，选择互联网行业是一个不错的选择，**有机会赚到大钱**。

# 互联网的缺点
1. 缺乏社会资源，通常只和计算机以及公司内部人员打交道，职业辐射范围太小。
2. 996比较常态化，缺乏时间照顾家庭。
3. 35岁是存在一定的职业危机，但其实也不用那么焦虑，重点在于：1.技术持续更新； 2.控制好自己的负债。 即使被裁员大不了降薪换个小点的公司。再强调一次，一定要控制好自己的负债。**普通人职业生涯不是直线**，大部分人是抛物线。![1717040027324-ee05f5db-407b-44d1-9213-1226f3aa76c4.png](./img/dK7iJdMwmELE-AZE/1717040027324-ee05f5db-407b-44d1-9213-1226f3aa76c4-065652.png)，**<font style="color:#DF2A3F;">所以一定要控制好自己的负债。</font>**
