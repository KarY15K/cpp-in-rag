# 22.查看磁盘、cpu 占用、内存占用命令


1.查看磁盘占用命令：



Linux系统：df -h



Windows系统：dir /s



2.查看CPU占用命令：



Linux系统：top、htop



Windows系统：Task Manager（任务管理器）



3.查看内存占用命令：



Linux系统：free -m



Windows系统：Task Manager（任务管理器）
