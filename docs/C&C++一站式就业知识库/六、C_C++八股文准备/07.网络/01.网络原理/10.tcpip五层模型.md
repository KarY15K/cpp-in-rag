# 10.tcp/ip五层模型


1. 应用层（Application Layer）：该层为用户提供应用程序，如电子邮件、文件传输、远程登录等。常见的协议有 HTTP、FTP、SMTP 等。
2. 传输层（Transport Layer）：该层负责数据传输，并保证数据传输的可靠性。常见的协议有 TCP 和 UDP。
3. 网络层（Internet Layer）：该层主要解决数据在网络中的路由问题。常见的协议有 IP、ICMP、ARP 等。
4. 数据链路层（Data Link Layer）：该层主要解决相邻节点之间的通信问题，将比特流转换成帧进行传输。常见的协议有 Ethernet、WiFi 等。
5. 物理层（Physical Layer）：该层实现物理连接和比特流传输，如网线、光纤等。
