# 51.extern C有什么作用


extern "C"的主要作用就是为了能够正确实现C<u>代码调用其他C语言代码。加上extern "C"后，会指示编译器这部分代码按C语言语法进行编译，而不是C</u>的。



extern 是变量或函数的申明，告诉编译器在其它文件中找这个变量或函数的定义。



这里需要的是“声明”，不是“定义”！根据C++标准的规定，一个变量声明必须同时满足两个条件，否则就是定义：  
(1)声明必须使用extern关键字；(2)不能给变量赋初值  
extern int a; //声明



int a; //定义



int a = 0; //定义



extern int a =0; //定义
