# 21.引用和指针的区别


引用是对象的别名， 操作引用就是操作这个对象， 必须在创建的同时有效得初始化(引用一个有效的对象， 不可为NULL)， 初始化完毕就再也不可改变， 引用具有指针的效率， 又具有变量使用的方便性和直观性， 在语言层面上引用和对象的用法一样， 在二进制层面上引用一般都是通过指针来实现的， 只是编译器帮我们完成了转换。 之所以使用引用是为了用适当的工具做恰如其分的事， 体现了最小特权原则。
