# 15.多线程里线程的同步方式有哪些


1. 互斥锁（Mutex）：用于保护临界区，确保同时只有一个线程可以进入临界区。
2. 信号量（Semaphore）：控制同时访问特定资源的线程数量。
3. 条件变量（Condition Variable）：等待某些条件满足后再继续执行。
4. 屏障（Barrier）：所有线程都必须到达屏障点，才能继续执行下一步操作。
5. 读写锁（Read-Write Lock）：允许多个读操作并发进行，但是写操作必须互斥执行。
6. 自旋锁（Spin Lock）：在获取锁之前不断循环检查锁是否可用，避免上下文切换带来的开销。
7. 原子操作（Atomic Operation）：在单个CPU指令中完成数据的读取和修改，避免竞争条件。



这些同步方式都有各自的优缺点和适用场景，在实际使用中需要根据具体情况进行选择。
