# Linux常用命令

Linux 中最常用的命令及其详细使用范例，涵盖文件操作、系统管理、网络管理等方面。

---

## **1. 文件操作**
### **1.1 **`**ls**`** - 列出目录内容**
+ **功能**：列出目录中的文件和子目录。
+ **常用选项**：
    - `-l`：以长格式显示详细信息。
    - `-a`：显示所有文件（包括隐藏文件）。
    - `-h`：以人类可读的格式显示文件大小。

**范例：**

ls -lh /home

**输出**：

```plain
drwxr-xr-x 2 user user 4.0K Jan  1 12:34 Documents
drwxr-xr-x 2 user user 4.0K Jan  1 12:34 Downloads
```

---

### **1.2 **`**cd**`** - 切换目录**
+ **功能**：切换当前工作目录。

**范例：**

```plain
cd /var/log  # 切换到 /var/log 目录
cd ~         # 切换到用户主目录
cd ..        # 切换到上一级目录
```

---

### **1.3 **`**cp**`** - 文件或目录**
+ **功能**：文件或目录。
+ **常用选项**：
    - `-r`：递归目录。
    - `-i`：覆盖前提示。

**范例：**

```plain
cp file1.txt file2.txt          # 文件
cp -r dir1 dir2                 # 目录
cp -i file1.txt /backup/        # 文件到备份目录
```

---

### **1.4 **`**mv**`** - 移动或重命名文件**
+ **功能**：移动文件或目录，或重命名文件。

**范例：**

```plain
mv file1.txt file2.txt          # 重命名文件
mv file1.txt /backup/           # 移动文件到备份目录
mv dir1 dir2                    # 重命名目录
```

---

### **1.5 **`**rm**`** - 删除文件或目录**
+ **功能**：删除文件或目录。
+ **常用选项**：
    - `-r`：递归删除目录。
    - `-f`：强制删除，不提示。

**范例：**

```plain
rm file1.txt                    # 删除文件
rm -r dir1                      # 删除目录
rm -rf dir1                     # 强制删除目录
```

---

### **1.6 **`**mkdir**`** - 创建目录**
+ **功能**：创建新目录。
+ **常用选项**：
    - `-p`：递归创建目录。

**范例：**

```plain
mkdir new_dir                   # 创建目录
mkdir -p dir1/dir2/dir3         # 递归创建目录
```

---

### **1.7 **`**touch**`** - 创建空文件或更新文件时间戳**
+ **功能**：创建空文件或更新文件的时间戳。

**范例：**

```plain
touch file1.txt                 # 创建空文件
touch -c file1.txt              # 更新文件时间戳
```

---

### **1.8 **`**cat**`** - 查看文件内容**
+ **功能**：查看文件内容。

**范例：**

```plain
cat file1.txt                   # 查看文件内容
cat file1.txt file2.txt         # 查看多个文件内容
```

---

### **1.9 **`**more**`** 和 **`**less**`** - 分页查看文件内容**
+ **功能**：分页查看文件内容。
+ **常用选项**：
    - `more`：只能向下翻页。
    - `less`：可以上下翻页。

**范例：**

```plain
more file1.txt                  # 分页查看文件内容
less file1.txt                  # 分页查看文件内容（支持上下翻页）
```

---

### **1.10 **`**find**`** - 查找文件**
+ **功能**：在指定目录下查找文件。
+ **常用选项**：
    - `-name`：按文件名查找。
    - `-type`：按文件类型查找。

**范例：**

```plain
find /home -name "*.txt"        # 查找 /home 目录下所有 .txt 文件
find /var/log -type f           # 查找 /var/log 目录下的所有普通文件
```

---

## **2. 系统管理**
### **2.1 **`**ps**`** - 查看进程状态**
+ **功能**：查看当前进程状态。
+ **常用选项**：
    - `-e`：显示所有进程。
    - `-f`：显示完整格式。

**范例：**

ps -ef                          # 显示所有进程的完整信息

---

### **2.2 **`**top**`** - 实时查看系统状态**
+ **功能**：实时查看系统状态和进程信息。

**范例：**

top                             # 实时查看系统状态

---

### **2.3 **`**kill**`** - 终止进程**
+ **功能**：终止指定进程。
+ **常用选项**：
    - `-9`：强制终止进程。

**范例：**

```plain
kill 1234                       # 终止进程 ID 为 1234 的进程
kill -9 1234                    # 强制终止进程
```

---

### **2.4 **`**df**`** - 查看磁盘使用情况**
+ **功能**：查看文件系统的磁盘使用情况。
+ **常用选项**：
    - `-h`：以人类可读的格式显示。

**范例：**

df -h                           # 查看磁盘使用情况

---

### **2.5 **`**du**`** - 查看目录大小**
+ **功能**：查看目录或文件的大小。
+ **常用选项**：
    - `-h`：以人类可读的格式显示。
    - `-s`：显示总大小。

**范例：**

du -sh /home                    # 查看 /home 目录的总大小

---

### **2.6 **`**free**`** - 查看内存使用情况**
+ **功能**：查看系统内存使用情况。
+ **常用选项**：
    - `-h`：以人类可读的格式显示。

**范例：**

free -h                         # 查看内存使用情况

---

## **3. 网络管理**
### **3.1 **`**ping**`** - 测试网络连接**
+ **功能**：测试与目标主机的网络连接。

**范例：**

ping google.com                 # 测试与 google.com 的连接

---

### **3.2 **`**ifconfig**`** - 查看网络接口信息**
+ **功能**：查看和配置网络接口。

**范例：**

```plain
ifconfig                        # 查看所有网络接口信息
ifconfig eth0                   # 查看 eth0 接口信息
```

---

### **3.3 **`**netstat**`** - 查看网络状态**
+ **功能**：查看网络连接、路由表、接口统计信息。
+ **常用选项**：
    - `-t`：显示 TCP 连接。
    - `-u`：显示 UDP 连接。
    - `-n`：以数字形式显示地址和端口。

**范例：**

netstat -tun                    # 查看 TCP 和 UDP 连接

---

### **3.4 **`**ssh**`** - 远程登录**
+ **功能**：通过 SSH 协议远程登录到另一台主机。

**范例：**

ssh user@192.168.1.100          # 登录到 192.168.1.100

---

### **3.5 **`**scp**`** - 远程文件**
+ **功能**：通过 SSH 协议在本地和远程主机之间文件。

**范例：**

```plain
scp file.txt user@192.168.1.100:/home/user  # 文件到远程主机
scp user@192.168.1.100:/home/user/file.txt .  # 从远程主机文件
```

---

## **4. 压缩和解压**
### **4.1 **`**tar**`** - 打包和解包文件**
+ **功能**：打包和解包文件。
+ **常用选项**：
    - `-c`：创建打包文件。
    - `-x`：解包文件。
    - `-z`：使用 gzip 压缩。
    - `-v`：显示详细信息。

**范例：**

```plain
tar -czvf archive.tar.gz dir1   # 打包并压缩目录
tar -xzvf archive.tar.gz        # 解压并解包文件
```

---

### **4.2 **`**gzip**`** 和 **`**gunzip**`** - 压缩和解压文件**
+ **功能**：压缩和解压文件。

**范例：**

```plain
gzip file.txt                   # 压缩文件
gunzip file.txt.gz              # 解压文件
```

---

### **4.3 **`**zip**`** 和 **`**unzip**`** - 压缩和解压文件**
+ **功能**：压缩和解压文件。

**范例：**

```plain
zip archive.zip file1.txt file2.txt  # 压缩文件
unzip archive.zip                    # 解压文件
```

## **5. 文本处理**
### **5.1 **`**grep**`** - 查找文本**
+ **功能**：在文件或输入中查找匹配的文本。
+ **常用选项**：
    - `-i`：忽略大小写。
    - `-r`：递归查找目录中的文件。
    - `-v`：反向查找（不匹配的行）。

**范例：**

```plain
grep "error" log.txt            # 查找包含 "error" 的行
grep -i "error" log.txt         # 忽略大小写查找
grep -r "error" /var/log        # 递归查找 /var/log 目录中的文件
```

---

### **5.2 **`**sed**`** - 流编辑器****（先了解）**
+ **功能**：对文本进行流式编辑。
+ **常用选项**：
    - `s/old/new/`：替换文本。
    - `-i`：直接修改文件。

**范例：**

```plain
sed 's/foo/bar/' file.txt       # 将文件中的 "foo" 替换为 "bar"
sed -i 's/foo/bar/' file.txt    # 直接修改文件
```

---

### **5.3 **`**awk**`** - 文本处理工具****（先了解）**
+ **功能**：强大的文本处理工具，支持按列处理。

**范例：**

```plain
awk '{print $1}' file.txt       # 打印文件的第一列
awk -F: '{print $1}' /etc/passwd # 以冒号为分隔符，打印第一列
```

---

### **5.4 **`**cut**`** - 按列提取文本****（先了解）**
+ **功能**：按列提取文本。
+ **常用选项**：
    - `-d`：指定分隔符。
    - `-f`：指定列。

**范例：**

cut -d: -f1 /etc/passwd         # 提取 /etc/passwd 文件的第一列（用户名）

---

### **5.5 **`**sort**`** - 排序文本****（先了解）**
+ **功能**：对文本进行排序。
+ **常用选项**：
    - `-n`：按数值排序。
    - `-r`：逆序排序。

**范例：**

```plain
sort file.txt                   # 对文件内容排序
sort -nr file.txt               # 按数值逆序排序
```

---

### **5.6 **`**uniq**`** - 去重****（先了解）**
+ **功能**：去除重复的行。
+ **常用选项**：
    - `-c`：统计重复次数。

**范例：**

```plain
uniq file.txt                   # 去除重复行
sort file.txt | uniq -c         # 统计重复次数
```

uname -a                        # 显示所有系统信息

## **7. 用户和权限**
### **7.1 **`**useradd**`** 和 **`**userdel**`** - 添加和删除用户**
+ **功能**：添加或删除用户。

**范例：**

```plain
useradd new_user                # 添加用户
userdel old_user                # 删除用户
```

---

### **7.2 **`**passwd**`** - 修改用户密码**
+ **功能**：修改用户密码。

**范例：**

passwd user1                    # 修改 user1 的密码

---

### **7.3 **`**chmod**`** - 修改文件权限**
+ **功能**：修改文件或目录的权限。
+ **常用选项**：
    - `u/g/o/a`：用户/组/其他/所有。
    - `+/-`：添加/删除权限。
    - `r/w/x`：读/写/执行。

**范例：**

```plain
chmod u+x script.sh             # 给用户添加执行权限
chmod 755 file.txt              # 设置文件权限为 rwxr-xr-x
```

---

### **7.4 **`**chown**`** - 修改文件所有者****（先了解）**
+ **功能**：修改文件或目录的所有者。

**范例：**

chown user1:group1 file.txt     # 修改文件所有者和组

## **8. 包管理**
### **8.1 **`**apt**`** (Debian/Ubuntu)**
+ **功能**：包管理工具。
+ **常用命令**：
    - `install`：安装软件包。
    - `remove`：卸载软件包。
    - `update`：更新软件包列表。
    - `upgrade`：升级已安装的软件包。

**范例：**

```plain
sudo apt update                 # 更新软件包列表
sudo apt install nginx          # 安装 nginx
sudo apt remove nginx           # 卸载 nginx
```

---

### **8.2 **`**yum**`** (CentOS/RHEL)**
+ **功能**：包管理工具。
+ **常用命令**：
    - `install`：安装软件包。
    - `remove`：卸载软件包。
    - `update`：更新软件包。

**范例：**

```plain
sudo yum install nginx          # 安装 nginx
sudo yum remove nginx           # 卸载 nginx
sudo yum update                 # 更新软件包
```

## **9. 其他常用命令**
### **9.1 **`**wget**`** - 下载文件**
+ **功能**：从网络下载文件。

**范例：**

wget https://example.com/file.zip  # 下载文件


+ **功能**：打包和解包文件。
+ **常用选项**：
    - `-c`：创建打包文件。
    - `-x`：解包文件。
    - `-z`：使用 gzip 压缩。
    - `-v`：显示详细信息。

**范例：**

```plain
tar -czvf archive.tar.gz dir1   # 打包并压缩目录
tar -xzvf archive.tar.gz        # 解压并解包文件
```

---

**范例：**

```plain

0 2 * * * /path/to/script.sh
```
