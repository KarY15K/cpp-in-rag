# 2.7 VMware Ubuntu命令行字体很小-高分辨率电脑

随着电脑屏幕分辨率越来越高，不少初学者在使用VMware安装Ubuntu后，发现命令行的字体特别小，不容易看清楚字体，那该如何解决这个问题？

![1735008972175-efd69af6-b94b-471e-8b4d-782408916fa2.png](./img/QSKfrOD0HZj9CkyT/1735008972175-efd69af6-b94b-471e-8b4d-782408916fa2-383231.png)

作为晚上十二点的时候，远程一个同学帮他解决了这个问题。

这个图是我远程的时候从我桌面截取他的VMware的设置，因为从低分辨率屏幕截取高分辨率屏幕，所以这张图不是太清晰。

![1735009166610-7e089dbd-c549-4578-895a-491fbc1b56ae.png](./img/QSKfrOD0HZj9CkyT/1735009166610-7e089dbd-c549-4578-895a-491fbc1b56ae-805605.png)



我给大家讲解下具体步骤：

1. 先在VMware菜单选项： 虚拟机 -> 电源 -> 关闭客户机

![1735009640464-73c71e6e-e312-4c3c-b0fe-e68979c32093.jpeg](./img/QSKfrOD0HZj9CkyT/1735009640464-73c71e6e-e312-4c3c-b0fe-e68979c32093-323367.jpeg)



2. 然后在编辑对应Ubuntu系统的：虚拟机设置-> 显示器 
    1. 监视器 里选择 “指定监视器设置”，“任意监视器的最大分辨率”选择 >= 屏幕的分辨率。
    2. 勾选拉伸模式，可以选择 “自由拉伸”。

![1735009166610-7e089dbd-c549-4578-895a-491fbc1b56ae.png](./img/QSKfrOD0HZj9CkyT/1735009166610-7e089dbd-c549-4578-895a-491fbc1b56ae-269348.webp)



3. 然后再重启虚拟机，字体就很大了，但要注意，一般我们都是用ssh工具连接Ubuntu进行命令操作，比如xshell工具。
