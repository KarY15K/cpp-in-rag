# 2.4 Ubuntu无法联网：ifconfig查看只有lo,没有ens33问题解决参考方法

问题：

在Ubuntu系统下，浏览器无法上网，终端输入ifconfig查看，只能看到lo本地回环网卡，没有找到ens33网卡。



原因：

这种情况出现一般是在上次正常使用可以上网，之后挂起或关机后再重启系统出现，初步判定与网卡状态文件记录有关 。（在Linux下，一切皆文件）。



**解决方法1：**

首先停止网络服务：

```markdown
service network-manager stop
```

删除设备网卡状态管理文件：

```bash
sudo rm -rf /var/lib/NetworkManager/NetworkManager.state 
```

<font style="color:rgb(38, 38, 38);">重新启动网络服务，这时网络状态将会重新加载刷新写入到文件：</font>

```bash
service network-manager start
```

<font style="color:rgb(38, 38, 38);background-color:rgb(250, 250, 250);"></font>

**<font style="color:rgb(38, 38, 38);">解决办法2：</font>**

```bash
sudo dhclient ens33
sudo ifconfig ens33
```
