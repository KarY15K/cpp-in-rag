# 2.6 两台虚拟机，一台有网，一台没网

**<font style="color:rgb(77, 77, 77);">原因是：</font>**

**<font style="color:rgb(77, 77, 77);">mac绑定的ip地址冲突了。俩个虚拟机的mac地址。在copy虚拟机打开的时候，没有点击【我已复制虚拟机】，而是选择了【我已移动虚拟机】</font>**

**<font style="color:rgb(77, 77, 77);">解决办法是：</font>**<font style="color:rgb(77, 77, 77);">  
</font>**<font style="color:#DF2A3F;">虚拟机关机</font>**<font style="color:rgb(77, 77, 77);">->编辑虚拟机设置->网络适配器->点击【生成】->确定好，开机了就都有网了。</font>

<font style="color:rgb(77, 77, 77);">只需要改一台就行，下次记得</font>**<font style="color:rgb(77, 77, 77);">选择我已复制</font>**<font style="color:rgb(77, 77, 77);">，这样不同的虚拟机mac地址才会有区别，才不会起冲突。
