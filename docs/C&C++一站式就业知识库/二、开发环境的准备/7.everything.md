# <font style="color:rgb(13, 11, 34);">1.限定文件夹</font>
<font style="color:rgb(13, 11, 34);">我要搜索一个文件，却发现搜出来的文件有成百上千个，还是没办法快速找到我想要的文件，但我知道它是被放到D盘的下载文件夹，这时就可以在前边先输入它的路径加一个空格再输入文件名，这样就可以只搜索某个文件夹的内容</font>

![1713534001521-7ca663e2-18d7-4360-9fcc-c1ad13fa4576.png](./img/44549646_Co8untvIjr_ByaAY/1713534001521-7ca663e2-18d7-4360-9fcc-c1ad13fa4576-011219.png)

<font style="color:rgb(13, 11, 34);"></font>

<font style="color:rgb(13, 11, 34);"></font>

<font style="color:rgb(13, 11, 34);">除了限定只搜索某个文件夹，还可以在文件夹路径前加一个英文感叹号，这样Everything就会在搜索时，排除掉这个文件夹中的内容，或者在设置的“排查搜索”选项中，永久排除掉某个文件或者文件夹的搜索</font>

![1713534013001-3e031a82-ed4b-445c-8c91-7dad04795120.png](./img/44549646_Co8untvIjr_ByaAY/1713534013001-3e031a82-ed4b-445c-8c91-7dad04795120-385240.png)

# <font style="color:rgb(13, 11, 34);">2.运算符</font>
<font style="color:rgb(13, 11, 34);">上面提到的英文感叹号其实是程序中常用的非运算符，表达否定的意思。类似的Everything中还有另外两个预算符与和或。我们在搜索框中输入abc空格123，此时会触发与，Everything会搜索文件名中包含abc和123的文件</font>

![1713534027236-3d28fb04-7896-4548-ab52-ece5a82aab82.png](./img/44549646_Co8untvIjr_ByaAY/1713534027236-3d28fb04-7896-4548-ab52-ece5a82aab82-510061.png)

<font style="color:rgb(13, 11, 34);"></font>

<font style="color:rgb(13, 11, 34);"></font>

<font style="color:rgb(13, 11, 34);">输入abc | 123会触发或，只要文件名中包含 abc 或 123 中的任意一个就会被搜索出来</font>

![1713534033827-88a504e5-ff66-4b34-9f4b-ed6053243ff2.png](./img/44549646_Co8untvIjr_ByaAY/1713534033827-88a504e5-ff66-4b34-9f4b-ed6053243ff2-847978.png)

# <font style="color:rgb(13, 11, 34);">7 总结Everything 功能特色</font>
<font style="color:rgb(13, 11, 34);">1.可以快速索引并搜索计算机上存储的所有文件和文件夹，搜索结果准确、快速。</font>

<font style="color:rgb(13, 11, 34);">2.具有小巧、便捷的特点，安装文件非常小，并且使用非常简单。</font>

<font style="color:rgb(13, 11, 34);">3.采用实时更新技术，搜索结果在创建或修改文件时立即更新，这意味着搜索结果永远是最新的。</font>

<font style="color:rgb(13, 11, 34);">4.不仅支持基本的文件名搜索，还支持高级文件名搜索，如通配符（*和？）、正则表达式等搜索方式，可以快速定位需要的文件。</font>

<font style="color:rgb(13, 11, 34);">5.用户可以使用多种条件设置过滤器，以限制搜索结果的范围，例如按文件类型、大小、日期等属性过滤。</font>

<font style="color:rgb(13, 11, 34);">6.支持多种排序方式，比如按名称、大小、日期等标准排序，还支持自定义排序方式。</font>

<font style="color:rgb(13, 11, 34);">7.可以通过LAN（局域网）搜索其他网络计算机上的文件，支持对共享文件夹的搜索和管理。</font>

<font style="color:rgb(13, 11, 34);">8.可以轻松地分享文件和文件夹给他人，只需要将搜索结果复制到剪贴板即可。</font>

<font style="color:rgb(13, 11, 34);">9.支持多种外观设置，例如修改字体、颜色等，同时也支持多种语言和插件扩展。</font>

<font style="color:rgb(13, 11, 34);">10.还支持多种搜索方式，包括模糊搜索、全词搜索、大小写敏感搜索等。</font>

<font style="color:rgb(13, 11, 34);">11.可以通过自动运行功能，使 Everything 开机自动启动并开始索引文件，以便更快地进行搜索。</font>

<font style="color:rgb(13, 11, 34);">12.可以快速访问最近使用的文件和文件夹，同时还可以收藏最常用的文件和文件夹。</font>

<font style="color:rgb(13, 11, 34);">13.用户可以使用JavaScript编写脚本，并在 Everything 中执行，从而实现更高级的搜索和过滤功能。</font>

<font style="color:rgb(13, 11, 34);">14.能够记录用户的搜索历史，可以随时查看和编辑之前的搜索请求。</font>

<font style="color:rgb(13, 11, 34);">15.支持许多插件，用户可以根据需要安装和配置插件来实现更多的功能，如复制文件路径、打开 CMD 窗口等等。</font>

<font style="color:rgb(13, 11, 34);">16.提供了交互式的界面设计工具，可视化设计规则来过滤搜索结果。</font>

<font style="color:rgb(13, 11, 34);">17.内置文件浏览器，可以直接查看和操作搜索结果中的文件，避免了打开资源管理器的繁琐操作。</font>
